# Changelog — 2026-06-21

## Version 0.1.0 — Initial MVP Release

### Added

- **Core Engine**
  - Mission Intake module with input validation and crop profile lookup
  - Environment Analyzer with weather assessment and flight condition determination
  - Swarm Planner with grid-based field partitioning and drone assignment
  - Route Planner with boustrophedon path generation and distance estimation
  - Resource Planner with battery, liquid, refill, and duration estimation
  - Risk Engine evaluating weather, battery, coverage, and operational risks
  - Decision Engine generating GO/NO-GO recommendations with confidence scoring

- **User Interface (Streamlit)**
  - Mission configuration sidebar with field, drone, and weather parameters
  - Swarm allocation visualization with interactive sector map
  - Route preview with boustrophedon flight paths
  - Resource dashboard with battery, liquid, and timeline charts
  - Risk radar visualization and detailed risk cards
  - Final recommendation panel with mission summary

- **Configuration**
  - Centralized settings with dataclass-based configuration
  - 8 crop profiles (wheat, corn, rice, soybean, vineyard, cotton, sugarcane, generic)
  - Environment variable support via .env files

- **Documentation**
  - Project vision document
  - Architecture v1 specification
  - ADR-001: Repository structure
  - ADR-002: Streamlit choice
  - ADR-003: Modular architecture
  - Roadmap through v2.0

- **Deployment**
  - requirements.txt with pinned dependencies
  - Dockerfile for containerized deployment
  - .env.example configuration template
